# pfun-health/pfun-data

    NOTE: under very active development!

Data utilities for PFun Digital Health.

## CLI Usage

```bash

$ pfun-data-unzipcsv2parquet \
    --zip-path $PWD/_raw_data/d1namo-ecg-glucose-data.zip \
    --csv-path $PWD/_raw_data/d1namo-ecg-glucose-data/csv_files \
    --parquet-path $PWD/_raw_data/d1namo-ecg-glucose-data/parquet_files

```
